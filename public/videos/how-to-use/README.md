Use the section ID as the filename:
- `getting-started.mp4`
- `quiz-modes.mp4`
- `question-types.mp4`
- `keyboard-controls.mp4`
- `custom-units.mp4`
- `quiz-features.mp4`
- `troubleshooting.mp4`
- `tips.mp4`
